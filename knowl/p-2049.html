At the start of the second pass, the largest value is now in place. There are \(n-1\) items left to sort, meaning that there will be \(n-2\) pairs. Since each pass places the next largest value in place, the total number of passes necessary will be \(n-1\text{.}\) After completing the \(n-1\) passes, the smallest item must be in the correct position with no further processing required.
