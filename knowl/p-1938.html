A variation of the linear probing idea is called quadratic probing. Instead of using a constant “skip” value, we use a rehash function that increments the hash value by 1, 3, 5, 7, 9, and so on. This means that if the first hash value is h, the successive values are \(h+1\text{,}\) \(h+4\text{,}\) \(h+9\text{,}\) \(h+16\text{,}\) and so on. In other words, quadratic probing uses a skip consisting of successive perfect squares. Figure 6.5.11 shows our example values after they are placed using this technique.
