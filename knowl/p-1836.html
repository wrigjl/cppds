In C++, the standard library provides, the find function which works on several C++ data types (e.g. arrays, vector, etc. ListingÂ 6.2.1 shows how find can be used to search an array or a vector. The first argument to find is the base object to search, the second argument is the element past the end (this is how find knows where to stop), and the third argument value to find is the item to locate. If the item is not found, the second argument is returned. If it is found, an iterator is returned.
