In order to code the algorithm in C++, we will use a hash map called prec to hold the precedence values for the operators which will be implemented with an unordered map. This hash map will map each operator char to an integer that can be compared against the precedence levels of other operators (we have arbitrarily used the integers 3, 2, and 1). The left parenthesis will receive the lowest value possible. This way any operator that is compared against it will have higher precedence and will be placed on top of it.
