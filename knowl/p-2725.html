Let's look at how the bfs function would construct the breadth first tree corresponding to the graph in Figure 9.8.1. Starting from fool we take all nodes that are adjacent to fool and add them to the tree. The adjacent nodes include pool, foil, foul, and cool. Each of these nodes are added to the queue of new nodes to expand. Figure 9.9.1 shows the state of the in-progress tree along with the queue after this step.
