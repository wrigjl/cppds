You can probably think of a couple of easy ways to implement a priority queue using sorting functions and vectors. However, inserting into a vector is \(O(n)\) and sorting a vector is \(O(n \log{n})\text{.}\) We can do better. The classic way to implement a priority queue is using a data structure called a binary heap. A binary heap will allow us both enqueue and dequeue items in \(O(\log{n})\text{.}\)
