Path

A path in a graph is a sequence of vertices that are connected by edges. Formally we would define a path as \(w_1, w_2, ..., w_n\) such that \((w_i, w_{i+1}) \in E\) for all \(1 \le i \le n-1\text{.}\) The unweighted path length is the number of edges in the path, specifically \(n-1\text{.}\) The weighted path length is the sum of the weights of all the edges in the path.
