Cycle

A cycle in a directed graph is a path that starts and ends at the same vertex. For example, in FigureÂ 9.2.1 the path \((V5,V2,V3,V5)\) is a cycle. A graph with no cycles is called an acyclic graph. A directed graph with no cycles is called a directed acyclic graph or a DAG. We will see that we can solve several important problems if the problem can be represented as a DAG.
