A hash table is a collection of items which are stored in such a way as to make it easy to find them later. Each position of the hash table, often called a slot, can hold an item and is named by an integer value starting at 0. For example, we will have a slot named 0, a slot named 1, a slot named 2, and so on.Initially, the hash table contains no items so every slot is empty. We can implement a hash table by using a list with each value initialized to an empty string.
