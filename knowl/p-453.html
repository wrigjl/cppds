Listing 1.13.11 adds the Connector class. The two gate instances within each connector object are referred to as the fromgate and the togate, recognizing that data values will “flow” from the output of one gate into an input line of the next. The call to setNextPin is very important for making connections. We need to add this method to our gate classes so that each togate can choose the proper input line for the connection.
