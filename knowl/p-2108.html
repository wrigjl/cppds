We said earlier that the way in which the increments are chosen is the unique feature of the shell sort. The function shown in Task 7.6.1.a uses a different set of increments. In this case, we begin with \(\frac {n}{2}\) subvectors. On the next pass, \(\frac {n}{4}\) subvectors are sorted. Eventually, a single vector is sorted with the basic insertion sort. Figure 7.6.4 shows the first subvectors for our example using this increment.
